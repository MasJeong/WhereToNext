## 변경 내용

- 전역 token 시스템을 warm amber 중심 색상에서 white + green + blue 팔레트로 재작업했다.
- 더 강한 브랜드 인지를 위해 주요 action token을 green으로 설정했다.
- 선택 상태를 amber 대신 mint/green 계열로 옮겼다.
- trust/info surface를 warm ivory 대신 pale blue 계열로 옮겼다.
- shared shell brand mark와 shared shell note surface를 새 팔레트에 맞게 갱신했다.
- 공통 shared card, action button, muted surface, stage shell을 갱신해 주요 경험 전반에 새 색상이 퍼지도록 했다.
