# 한눈에 보기
- 결과 페이지 범위에서 `nature`를 `아웃도어`로 바꿨다.
- 요약 카드와 결과 설명, 필터 카피를 활동/장면 중심으로 정리했다.
- YouTube 카드 영역은 그대로 두었다.

## 조사 요약
- Tripadvisor는 주요 분류어로 `Outdoor Activities`를 사용한다.
- GetYourGuide도 `Outdoor activities`를 상위 분류로 두고 그 아래 `Hiking`, `Nature`, `Surfing` 같은 세부 활동을 둔다.
- Airbnb Experiences는 `Nature and outdoors`처럼 `nature`를 단독 명사보다 `outdoors`와 결합해 활동 문맥으로 보여준다.

## 적용 내용
- 결과 페이지용 vibe 라벨 헬퍼를 따로 만들고 `nature -> 아웃도어`로 치환했다.
- 결과 페이지 상단 태그와 `다른 후보` 카드 태그에 결과 전용 라벨을 사용하도록 바꿨다.
- 결과 요약 카드의 `핵심 분위기` 값도 결과 전용 라벨을 쓰도록 정리했다.
- 결과 필터의 `쉬는 흐름` 표현을 `아웃도어`로 바꾸고 설명도 `해변·풍경` 중심으로 바꿨다.
