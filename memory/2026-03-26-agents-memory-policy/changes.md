# 변경 사항

## 파일
- `AGENTS.md`: 큰 작업 시 `memory/` 아래 공개 작업 기록을 남기도록 규칙을 추가했다.
- `memory/README.md`: `memory/` 폴더의 목적, 생성 기준, 경로 규칙, 기본 템플릿을 문서화했다.

- `memory/2026-03-26-agents-memory-policy/changes.md`: 이번 작업의 실제 변경 파일을 기록했다.

- `memory/`는 사용자가 검토할 수 있는 작업 기록용이고, `.sisyphus/`는 내부 스크래치 공간으로 분리했다.
