# 검증

## 명령 / 확인 사항
- `AGENTS.md` 읽기
- `memory/README.md` 읽기
- `docs/issue-resolution-log.md` 읽기

## 결과
- `AGENTS.md`에 `memory/` 규칙이 추가되었고, 큰 작업의 공개 작업 기록 경로와 최소 파일 구성이 명시됐다.
- `memory/README.md`가 생성되어 폴더 목적, 예시 경로, 템플릿, 주의사항을 확인할 수 있다.
- `docs/issue-resolution-log.md`와의 역할 구분도 문서에 반영되어 기록 목적이 충돌하지 않는다.

## 남은 리스크
- 앞으로 어떤 작업을 `큰 작업`으로 볼지에 대한 팀 해석 차이는 남을 수 있다.
- 실제 대형 작업에서 이 규칙이 꾸준히 지켜지는지는 후속 작업으로 확인해야 한다.
