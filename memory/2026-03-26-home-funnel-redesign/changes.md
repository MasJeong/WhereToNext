# 변경 사항

- `landing-page.tsx`를 간결한 한국어 카피와 중앙 정렬 단일 CTA 구조의 랜딩으로 재작업했다.
- `hero-animation.tsx`를 reduced-motion-safe CSS animation hook을 갖춘 미니멀 단색 일러스트로 재구성했다.
- `step-question.tsx`와 `progress-bar.tsx`를 화면당 질문 1개 간격과 단순한 진행 바로 맞춰 단순화했다.
- `result-page.tsx`를 대표 목적지 대형 요약, 태그, 간결한 사실 정보, 더 가벼운 액션 레이아웃을 지원하도록 확장했다.
- `home-experience.tsx`의 결과 렌더링을 재구성해 추천 로직은 유지하면서 촘촘한 카드를 대표 영역 + 간결한 보조 목록과 기존 저장, 공유, 비교 동작으로 교체했다.
- `globals.css`에 흰색 중심 퍼널 토큰과 작은 애니메이션 규칙을 추가했다.
