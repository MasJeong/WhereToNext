# 한눈에 보기

- 목표: 여행 기록을 더 빨리 등록하게 하고, 기존 기록도 리스트에서 바로 수정할 수 있게 만든다.
- 범위: 기록 카드 수정 액션, edit route, step 입력 가속화, 메모/날짜 빠른 제안, 관련 검증까지 포함한다.
- 기준: Polarsteps의 step 수정 가능 흐름과 Apple Journal의 제안 기반 입력 시작 방식을 참고한다.

## 계획

1. 현재 기록 추가/리스트 흐름에서 수정 진입과 느린 입력 지점을 찾는다.
2. 리스트 카드에 수정 진입을 추가하고 edit route를 만든다.
3. step 입력에 빠른 선택과 자동 진행을 넣어 등록 속도를 높인다.
4. 브라우저 테스트로 생성/수정 흐름을 확인한다.
