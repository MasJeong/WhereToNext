# 한눈에 보기
- 우측 컬럼에서 중복 설명과 부연 문장을 더 줄였다.
- fact 카드는 값 중심으로 바꾸고, 행동 카피를 더 짧게 정리했다.
- YouTube 카드 영역은 그대로 유지했다.

## 적용 내용
- 리드 설명 문장을 2줄 안쪽으로 유지해 제목과 추천 이유가 먼저 읽히게 했다.
- `추천 시기/예산/비행 거리` fact 행에서 상세 설명을 제거해 스캔 속도를 높였다.
- CTA 카피를 `일정 담기`, `다시 고르기`처럼 더 짧고 직접적인 표현으로 바꿨다.
- 우측 컬럼 전체를 `핵심 이유 -> 태그 -> fact -> 행동` 순서로 더 또렷하게 읽히도록 정리했다.
