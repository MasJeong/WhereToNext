# 한눈에 보기

- 생성 중 화면이 향후 광고 슬롯을 받을 수 있도록 문서 전제를 남겼다.
- 핵심은 `결과와 분리된 생성 단계`, `작은 슬롯`, `명확한 스폰서 표기`다.

## 문서 반영 내용

### 1. 안전한 광고 지면 추가
- `docs/uiux-style-guide.md`
- 기존 안전 지면 목록에 `추천 결과 직전 생성 중 전용 화면의 소형 파트너 슬롯`을 추가했다.

### 2. 생성 중 화면 전용 규칙 추가
- `docs/uiux-style-guide.md`
- 광고는 로딩바 위 보조 슬롯이나 옆 소형 슬롯으로만 두고, 모바일은 본문 아래 소형 카드 1개까지만 허용하도록 적었다.
- 생성 중 광고는 결과 메시지보다 더 강하게 보이면 안 되고, 여행 준비 맥락의 제휴만 허용한다고 명시했다.
