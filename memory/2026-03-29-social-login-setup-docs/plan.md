# 한눈에 보기

- 목표: 현재 붙어 있는 Google, Kakao, Apple 소셜 로그인이 실제 운영/검증 환경에서도 동작하도록 필요한 설정값과 절차를 문서화한다.
- 범위: 코드 기준 필수 환경 변수, callback URL 규칙, 공급자 콘솔 설정, 로컬 mock/실공급자 테스트 방법, 운영 체크리스트를 정리한다.
- 제외: 실제 공급자 콘솔에 값을 대신 입력하거나 비밀값을 생성하는 작업은 문서 범위 밖이다.

## 계획

1. OAuth 구현 코드를 읽고 실제로 필요한 환경 변수와 callback 구조를 확인한다.
2. 공급자 공식 문서 기준으로 Google, Kakao, Apple 설정 절차를 대조한다.
3. 저장소에 한국어 설정 가이드를 추가하고 README에서 연결한다.
4. 문서 경로와 핵심 내용을 memory 기록으로 남긴다.
