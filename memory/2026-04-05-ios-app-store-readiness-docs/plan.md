- iOS 배포를 위한 기존 체크리스트 문서가 있었고, App Store 정책 관점이 부족했다.
- 이번 작업은 기존 체크리스트를 보강하고, 공개용 실행 계획 문서를 추가하는 것이다.

# iOS App Store 준비 문서 정리

## 목표
- 기존 iOS 출시 점검 문서를 App Store 심사/정책 항목까지 확장한다.
- 내부 계획과 별도로 사용자가 바로 볼 수 있는 공개 계획 문서를 만든다.

## 범위
- 포함 범위:
  - 기존 `docs/ios-release-preflight.md` 검토 및 보강
  - 공개용 `docs/ios-app-store-readiness-plan.md` 작성
  - 작업 기록 `memory/` 추가
- 제외 범위:
  - 실제 기능 구현
  - Xcode / TestFlight 실행
  - 정책 항목의 법률 자문 확정

## 계획
1. 기존 iOS 관련 문서와 내부 계획 문서를 확인한다.
2. 공식 Apple 조사 결과를 기준으로 누락된 체크 항목을 정리한다.
3. 체크리스트 문서와 공개 계획 문서를 작성한다.
4. 작업 기록과 검증 결과를 남긴다.

## 리스크
- 실제 심사 통과 여부는 구현 상태와 App Store Connect 입력값에 따라 달라진다.
- 외부 콘텐츠 권리와 privacy disclosure는 추후 운영 정책 문서와 함께 더 구체화해야 한다.
