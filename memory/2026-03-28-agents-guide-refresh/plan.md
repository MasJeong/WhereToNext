# 한눈에 보기

- 상태: 진행 중
- 목적: 루트 `AGENTS.md`를 저장소 실상에 맞는 한국어 에이전트 가이드로 정리
- 범위: 명령어, 단일 테스트 실행, 코드 스타일, 타입/검증/테스트 규칙, `.sisyphus`와 `memory` 역할

## 작업 계획

1. 실제 config와 기존 `AGENTS.md`를 읽고 현재 규칙 스택을 확인한다.
2. `package.json`, `vitest.config.ts`, `playwright.config.ts`에서 실행 명령과 단일 테스트 패턴을 추출한다.
3. 대표 source/config 파일을 근거로 스타일/타입/라우트/검증 규칙을 정리한다.
4. 루트 `AGENTS.md`를 한국어 중심 문서로 재작성한다.
5. 변경 내용을 검토하고 검증/리뷰 결과를 `memory/`에 남긴다.
