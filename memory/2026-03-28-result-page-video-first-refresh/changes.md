# 한눈에 보기
- 상단 hero 이미지를 제거하고, 대표 YouTube 카드가 먼저 보이는 구조로 바꿨다.
- 우측 텍스트는 도시명, 추천 이유, 태그, 핵심 fact, CTA만 남겨 퍼널을 단순화했다.
- `다른 후보` 리스트는 더 촘촘하고 카드 중심으로 정리했다.

## 변경 내용
- `ResultPage`의 상단 2단 레이아웃 역할을 재정의했다.
  좌측: 대표 영상/placeholder 카드
  우측: 도시명, 추천 이유, 태그, fact 카드, 행동 영역
- `LeadSocialVideoPanel`을 큰 썸네일 중심 카드로 재설계했다.
  텍스트를 별도 열에서 설명하던 구조를 제거하고, 썸네일 위에 제목과 메타를 겹쳐 노출한다.
- `TravelSupportPanel`은 상단 대표 영역에서 분리해 아래 섹션으로 이동했다.
- `CompactRecommendationItem`의 카드 밀도와 간격을 정리해 `다른 후보` 리스트가 더 간결하게 보이도록 조정했다.
- 홈 페이지 빌드 시 `useSearchParams()` 제약을 맞추기 위해 `src/app/page.tsx`에 `Suspense` 경계를 추가했다.

## 영향 범위
- 대표 추천 상단 첫 fold의 정보 구조
- 소셜 비디오 카드의 시각 우선순위
- 결과 페이지 하단 `다른 후보` 리스트의 밀도와 정렬
