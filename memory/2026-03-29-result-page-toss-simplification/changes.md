# 한눈에 보기
- 우측 컬럼을 `설명`보다 `결정` 중심으로 다시 정리했다.
- 제목 위 라벨을 제거하고, 추천 설명은 2줄로 줄였다.
- fact는 3개 카드에서 3개 행으로 바꿨고, CTA는 2개 우선 구조로 단순화했다.

## 참고한 방향
- 토스는 공식 커뮤니케이션에서 사용자가 복잡한 정보도 `쉽게 조회`하고 `한 곳에서` 이해하도록 만드는 방향을 반복해서 강조한다.
- 그래서 결과 페이지 우측 컬럼도 설명을 늘리기보다, 핵심 판단과 행동을 앞세우는 구조가 더 적합하다고 봤다.

## 적용 내용
- `ResultPage` 우측 상단의 `대표 여행지` 라벨을 제거했다.
- `leadDescription`은 2줄까지만 보이도록 줄였다.
- `leadFacts`는 카드형 3개에서 행형 3개로 바꿨다.
- `바로 움직이기` 제목을 제거하고, `상세 보기`와 `내 일정에 담기`만 상단 CTA로 유지했다.
- `질문 다시 보기`, `처음부터 다시 하기`, 공유 액션은 하위 텍스트 액션으로 내렸다.
