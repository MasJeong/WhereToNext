# 상단 영역 목업 2차 시안

## 한눈에 보기
- 이전 박스·패널 중심 상단 시안을 폐기하고, 장면형 히어로 2종으로 다시 설계한다.
- 목표는 헤더+히어로가 하나의 연속된 표면처럼 읽히는 landing top area다.
- 산출물은 한국어 카피만 사용하는 SVG 2종과 설계 메모다.

## 목표
- 카드, 패널, 박스 설명 UI 없이도 바로 이해되는 상단 영역을 만든다.
- 헤더는 텍스트 중심 이동 구조로 단순하게 유지한다.
- 히어로는 여행 장면이 먼저 보이고, 큰 제목·짧은 설명·CTA 1개만 남긴다.

## 범위
- 포함: 상단 영역 replacement mockup SVG 2종, 구조 설명, 사용자 흐름, 비생산용 React 구조 참고 코드.
- 제외: production React 코드 수정, 기존 페이지 동작 변경, 보조 카드/라벨/패널 추가.

## 계획
1. 기존 톤·카피·디자인 토큰을 기준으로 scene-first 구조를 재정의한다.
2. no-card / no-panel / no-box 조건을 지키는 SVG 2종을 만든다.
3. 변경 이유와 구조 비교를 memory 문서에 남긴다.

## 리스크
- SVG 안에서 설명 요소가 다시 라벨처럼 보이지 않게 절제해야 한다.
- CTA 외 강조 요소가 생기면 one-purpose 원칙이 흐려질 수 있다.
