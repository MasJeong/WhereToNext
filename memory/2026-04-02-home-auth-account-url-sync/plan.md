# 한눈에 보기

- 홈 퍼널, 인증 우회, 계정 탭의 상태 복원성을 함께 정리한다.
- 목표는 새로고침, 뒤로 가기, 로그인 이탈 후 복귀에서도 사용자가 맥락을 잃지 않게 만드는 것이다.

## 작업 계획

1. 인증 화면의 `로그인 없이 계속 보기`가 안전한 `next`로 복귀하도록 수정한다.
2. 비로그인 헤더에서 중복 `내 여행` 진입점을 제거해 정보 구조를 단순화한다.
3. 홈 질문 퍼널의 단계와 응답을 URL과 동기화해 새로고침과 뒤로 가기를 복원 가능하게 만든다.
4. 질문 초기값을 빈 상태로 바꿔 숨겨진 기본 선택을 제거한다.
5. 계정 탭을 `?tab=`과 동기화해 복귀성과 공유성을 확보한다.
6. 관련 unit/e2e 회귀 테스트를 보강하고 전체 검증을 수행한다.
